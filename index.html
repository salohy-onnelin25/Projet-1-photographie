Bienvenue sur mon premier site

Ce projet marque le début de mon apprentissage du développement web. J’expérimente avec le HTML et le CSS pour comprendre comment créer une page claire, agréable et moderne. Sur ce site, je vais partager mes premières créations, mes découvertes et mes progrès étape par étape. L’objectif est d’apprendre en pratiquant et de bâtir petit à petit un vrai portfolio qui montre mon évolution dans le monde du web.
